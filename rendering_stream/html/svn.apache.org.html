Welcome to the Apache Software Foundation Subversion Server

The Apache Infrastructure Team provides version control for all ASF projects. Some projects use Subversion, and others prefer Git.

Learn more about how the ASF uses version control.

If you are looking for the Apache Subversion Project, you can find it subversion.apache.org.

Contact `users@infra.apache.org` if you have issues with the Apache Subversion repositories.
