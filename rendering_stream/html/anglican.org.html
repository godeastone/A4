This web site is for the Anglican (Episcopal) Church around the world. Its purpose is to help us Anglicans and Episcopalians communicate with each other more easily, and to help everyone learn about our church.

Use the menu on the left to look at various sections. If you are looking for the official web page of some part of the Anglican or Episcopal church, see the Official web pages section.

This Anglican Domain web site is not official in any way. It is produced and sponsored by the Society of Archbishop Justus as a global resource.

If you are here because of an IT-related issue (DNS changes, etc) please email technical@justus.anglican.org
