ОК для сайтов

OAuth авторизация

OAuth — открытый протокол авторизации, который позволяет безопасно авторизовать пользователя внешнего сайта через Одноклассники и получить доступ к API Одноклассников с сервера внешнего сайта или нативного приложения
Кнопка "Класс"

Кнопка “Класс” позволит посетителям вашего сайта отмечать особо понравившиеся материалы и делиться ими с друзьями в Одноклассниках
Виджет контента

При помощи данного виджета можно размещать любой контент из Одноклассников (топики, видео, фото, товары) на вашем сайте
Виджет группы

При помощи данного виджета можно рекламировать вашу группу на Одноклассниках. Он позволит посетителям вашего сайта присоединиться к ней буквально в один клик
Виджет профиля

При помощи данного виджета можно рекламировать ваш профиль на Одноклассниках. Он позволит посетителям вашего сайта подписываться на вас буквально в один клик
Виджет публикации

Для внешних приложений (например, сайтов или Android/iOS приложений) предусмотрен особый механизм публикации медиатопиков на страницу пользователя (эти записи увидят его друзья в своих лентах)
Виджет приглашения и рекомендации игр

Для внешних приложений (сайтов, нативных или же html5 мобильных игр) существует особый механизм привлечения в игру с помощью виджетов
Виджет видео

Целый ряд инструментов позволяет размещать видео из Одноклассников у вас на сайте, либо в приложении
